# OpenOCD Python API
Insert [for stmf103](ocd_config/cs32f1x.cfg) to targets
install [openocd xpack](https://openocd.org/pages/getting-openocd.html)

Run Server
```
bin\openocd  -s share\openocd -f interface/stlink-v2.cfg -f target/cs32f1x.cfg
```
